Social Media Etiquette Reminder

Block websites and webpages

Instruction to edit filter list:

  • Use only one line per website
  • To block a full website with all it's sub-domain and protocol like abc.com, en.abc.com, https://abc.com type *://*.abc.com/*
  • To block a specific sub-domain like en.abc.com, type *://en.abc.com/*
  • To block a specific webpage, like abc.com/attacker, type *://abc.com/attacker
  • To block a website/webpage with specific protocol, like https in abc.com, type https://abc.com/*
  • To block all protocol like http and https in abc.com, type *://abc.com/*
  • To block a specific file extension on all websites like .exe file, type *://*/*.exe
  • To block access of internet from this browser, type *://*/*

Basic syntax: [protocol]://[domain]/[path][protocol] can be http, https[domain] can be any valid domain name like abc.com, www.abc.com, en.abc.com, blog.abc.com, abc.co.uk[path] can be any specific URL path like https://abc.com/about-abc, https://abc.com/scam/best-scam.png

* can be used to define any protocol, domain, path like in *://*/* , first * means any protocol, second * means any domain, and third * means any path.

To unblock a website, remove its url from the filter list.
